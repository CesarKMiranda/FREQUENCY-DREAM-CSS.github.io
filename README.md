# PROYECTO FINAL MÓDULO 2

## PROYECTO "FREQUENCY DREAM"

Buen día, sean bienvenidos al repositorio de mi **proyecto final del Módulo 2**

Este documento sirve como guía para entender, estructurar y visualizar el proyecto **FREQUENCY DREAM**, una página web estática desarrollada como parte de mi proceso de aprendizaje en desarrollo web

---

## VISIÓN DEL PROYECTO

Este proyecto consiste en una **página web estática responsiva**, creada con el objetivo de aprender y aplicar los **fundamentos de HTML5 y CSS**

La página presenta a **FREQUENCY DREAM**, una escuela ficticia de diseño de sonido, mezcla y masterización. El sitio incluye:

- Un menú de navegación
- Sección informativa (Nosotros)
- Promociones
- Productos y cursos presentados en tarjetas
- Botones de acción (sin funcionalidad lógica aún)
- Pie de página con enlaces a redes sociales

A diferencia de la primera versión del proyecto, en esta etapa se incorporó **CSS** para mejorar el diseño visual, la organización del contenido y lograr una **experiencia responsiva**, adaptable a dispositivos móviles, tabletas y escritorio

El enfoque principal sigue siendo el aprendizaje, manteniendo un diseño limpio, entendible y realista, aplicando buenas prácticas como:
- Uso de etiquetas semánticas
- Separación de estructura (HTML) y estilos (CSS)
- Uso básico de Flexbox
- Diseño responsivo

---

## TECNOLOGÍAS UTILIZADAS

### HTML5
Utilizado para la estructura del sitio web, aplicando etiquetas semánticas como:

- `header`, `nav`, `main`, `section`, `article`, `footer`
- `h1`, `h2`, `h3`, `p`
- `ul`, `li`
- `img`, `a`, `button`

### CSS
Utilizado para el diseño visual y la responsividad del sitio:

- Flexbox para distribución de elementos
- Media Queries para diseño responsivo
- Variables CSS (`:root`)
- Hover effects y transiciones

> Nota: El proyecto **no utiliza JavaScript**, ya que el objetivo del módulo fue aprender la estructura y el diseño.

## INSTRUCCIONES DE USO
### 1. Requisitos

No se requiere software especializado. Solo necesitas un navegador web moderno:

- Google Chrome

- Mozilla Firefox

- Microsoft Edge

- Safari

### 2. Ejecución

Sigue estos pasos para ver el proyecto en funcionamiento:

- Descarga o clona este repositorio en tu computadora

- Asegúrate de que los archivos index.html y style.css se encuentren en la misma carpeta

- Abre el archivo index.html con doble clic o arrástralo a tu navegador

- El navegador renderizará automáticamente la página

- También puedes visitar la versión en línea del proyecto en GitHub Pages:

(https://cesarkmiranda.github.io/FREQUENCY-DREAM-CSS.github.io/)
